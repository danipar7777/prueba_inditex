# Prueba Inditex

Mini-aplicación para comprar dispositivos móviles hecha con ReactJS y Redux.

### Instalación 🔧

```
Bajar el proyecto
```
```
Ejecutar: npm install
```
```
Ejecutar:  npm start
```

## Ejecutando las pruebas ⚙️

```
Ejecutar: npm test
```

## Compilación 📦

```
Ejecutar: npm build
```
